## Javascript & DOM Lab

## A lab using Jest to test JavaScript

### Developer
Requires Node v10 or later.

### How to get started
* Fork repository, clone locally, navigate to repository directory,
* Download all the files with `npm i`,
* To test: `npm run test:watch`
